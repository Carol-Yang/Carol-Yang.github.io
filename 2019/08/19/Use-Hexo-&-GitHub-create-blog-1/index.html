雖然我是手寫筆記派，但是寫 blog 與 自己在紙本的筆記，還是有很大的不同，
紙本筆記比較像條列式寫下重點，而寫 blog 比較像是自己與自己的對話。

前陣子使用過 Medium 但覺得寫文章有諸多限制，最終決定使用 Hexo 作為我寫 blog 的地方。

話不多說，第一篇 blog 就從建立 blog 的過程開始吧!
